# AsesoriaLegal

Landing page profesional para **Lic. Valeria Montoya – Profesional del Derecho**, construida con **Vite + React**.

## Requisitos

- Node.js 18+
- npm 9+

## Instalación

```bash
npm install
```

## Scripts disponibles

| Comando            | Descripción                          |
| ------------------ | ------------------------------------ |
| `npm run dev`      | Inicia el servidor de desarrollo     |
| `npm run build`    | Genera la versión de producción      |
| `npm run preview`  | Vista previa del build de producción |
| `npm run lint`     | Ejecuta ESLint                       |
